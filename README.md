# Ironclad Construction - Professional Contracting Website

A bold, professional website for a construction and contracting company featuring an industrial design aesthetic with strong typography and clear service showcasing.

## Features

- **Bold Industrial Design** - Orange and blue color scheme with geometric elements
- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **Strong Typography** - Bebas Neue headlines paired with Roboto body text
- **Service Showcase** - Three main service categories with detailed lists
- **Statistics Display** - Animated stats showing company credentials
- **Trust Indicators** - Licensed, insured, and experienced messaging throughout
- **Multiple CTAs** - Strategic call-to-action placement for lead generation
- **Clean Navigation** - Fixed navigation with prominent quote button

## Sections

- **Hero Section** - Bold headline with dual CTAs and floating statistics
- **Services** - Three core service offerings (Commercial, Residential, Renovation)
- **Why Choose Us** - Company benefits with feature grid
- **CTA Section** - Prominent call-to-action for project inquiries
- **Contact** - Phone, email, and location information
- **Responsive Footer** - Company credentials and copyright

## Technologies Used

- HTML5
- CSS3 (Advanced animations and grid layouts)
- Google Fonts (Bebas Neue & Roboto)
- SVG patterns for background texture

## Design Philosophy

Industrial/utilitarian aesthetic emphasizing:
- Trust and reliability
- Professional craftsmanship
- Clear service communication
- Strong visual hierarchy
- Bold, confident branding

## Color Palette

- Primary Orange: #FF6B35 (Construction, energy)
- Dark Blue: #004E89 (Trust, professionalism)
- Steel Gray: #2E3440 (Industrial, strength)
- Light Gray: #D8DEE9 (Clean, modern)
